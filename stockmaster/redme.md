StockMaster 
A complete, full-stack inventory management system built with Node.js, Express, MongoDB, HTML, CSS, and vanilla JavaScript.

Install NodeJs
  bash#
  sudo apt install nodejs npm
  node -v
  
Step 1: Install MongoDB
  bash# On macOS
  brew tap mongodb/brew
  brew install mongodb-community
  brew services start mongodb-community
  # On Ubuntu/Debian
  sudo apt-get install mongodb

Step 2: Cloning the git repo
  bash#
  git clone https://github.com/HarshK2903/odoo.git
  cd odoo/stockmaster/
  
Step 3: Installing Packages
  bash#
  #for installation of all packages
  npm install dependencies

Step 4: Running the server
  bash#
  npm run dev

Step 5: 
  #Go to any browser and open 
  http://localhost:3000
